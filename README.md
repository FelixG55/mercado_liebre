
Proyecto intermedio del curso de Full Stack 
## Ejecutar localmente

Clonar el proyecto

```bash
  git clone https://github.com/FelixG55/mercado_liebre.git
```

Ir al directorio del proyecto

```bash
  cd mercado_liebre
```
Instalar dependencias 

```bash
  npm install
```

Iniciar apliacion

```bash
  npm run test
```
